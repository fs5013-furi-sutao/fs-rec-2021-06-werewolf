### 人狼ゲームとは？

人狼ゲームとは、  
味方になりすましたウソつきを会話で見つけ出すパーティーゲームです 

必要なのは・・・
コミュニケーション・観察・説得力

与えられたミッションは・・・
「人間たちの中にまぎれこんだ人狼を見つけ出せ！」🤔
